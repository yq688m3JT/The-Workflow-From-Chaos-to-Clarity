# Meeting Summarizer: From Chaos to Clarity

## Business Workflow
The goal of this process is to transform a 60-minute transcript or a mess of shorthand notes into a single source of truth that stakeholders can digest in under 60 seconds.

**Who the user is:** Project Managers, Team Leads, and busy Stakeholders who need to stay informed without attending every meeting.

**What input the system receives:** Raw meeting transcripts, messy shorthand notes, or bullet points from a discussion.

**What output the system should produce:** A structured summary including:
- High-level overview (1-2 sentences)
- Key Decisions Made
- Action Items (with owners if mentioned)
- Next Steps

**Why this task is valuable:** Capturing action items and decisions manually is time-consuming and prone to human error or bias. Automating the first draft ensures nothing is missed and provides a consistent format for all team communications, saving hours of administrative work.
